# 工具

## 1. [snip](https://mathpix.com)

Mathpix 出品的 Snip 是一个免费的的工具，可用把各种公式转换成 LaTeX。除了印刷的公式，手写公式也能识别转换，但是要求字迹工整。除了公式，化学方程式和矩阵也能识别，支持 Mac、Windows、Ubuntu。
