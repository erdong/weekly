# 工具

## 1. [DownGit](https://minhaskamal.github.io/DownGit/#/home)

一个小工具用来在 GitHub 上下载某个文件夹或者文件，还可以创建下载链接给其他人。

Github中并不提供单个文件夹下载, 每当下载仓库中某个文件夹时,只能克隆整个仓库, 浪费硬盘空间不说,浪费时间不说,每次克隆仓库时候,看到这下下载速度简直不能忍受，使用这个工具就好了。
